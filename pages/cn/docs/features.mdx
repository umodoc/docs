# 核心特性

Umo Editor 的特性太多太杂，很难简洁高效地表达清楚，不如在阅读之前访问[在线示例](https://demo.umodoc.com/editor)先体验一下，或许您对 Umo Editor 能有一个简单的认识。不过由于 CDN 服务器在海外，在线示例的加载速度可能稍慢。Nice to meet you！

Umo Editor 的特性包含但不限于以下几点：

### 零配置开箱即用
Umo Editor 是一个开源的 Vue3 文档编辑器，零配置开箱即用，同时又提供丰富的[配置项](./options/default)和友好实用的[事件](./events)、[方法](./methods)，开发者可以快速将 Umo Editor 集成到项目中，详见[快速开始](./guide/getting-started)。

### 类似于 Microsoft Word 的分页模式
Umo Editor 采用了类似 Microsoft Word 的分页模式，提供更好的编辑和阅读体验，并支持设置纸张大小、页边距、纸张方向和打印等功能。

### 轻量级
Umo Editor 核心 JS 文件大小仅约 1MB，其中已包含所有图标和图片（开启 gzip 后约 270KB），样式文件约 350KB（开启 gzip 后约 50KB）。其他外部依赖文件可按需引用。

不同版本的 NPM 包体积可能有所不同。

### 支持自定义扩展
Umo Editor 支持添加自定义扩展，允许开发者根据项目需求添加个性化的功能。无论是增加新的编辑命令，还是集成复杂的数据处理功能，自定义扩展都能让 Umo Editor 变得更加强大和灵活。

### 所见即所得
Umo Editor 的核心功能是所见即所得，用户从输入、保存到预览，再到打印，整个过程都是可控的，用户可以快速高效且专注地编辑文档。

### 富文本编辑
Umo Editor 提供基础且全面的富文本编辑功能，满足用户绝大部分场景下的文档编辑需求。

1. **基础格式**：支持设置不同级别的标题，以及段落、引用等多种类型的基础格式支持。
2. **文本样式**：可以对文本进行加粗、斜体、下划线和删除线等样式操作，同时支持文字颜色和背景色的自定义等。
3. **列表处理**：提供无序列表和有序列表，以及嵌套列表等用于组织文档内容。
4. **待办事项**：集成待办事项列表，方便用户创建和管理任务项。
5. **区块元素**：包括代码块、数学公式、分隔线、页面大纲、文本框、模板、网页等。
6. **链接与图片**：轻松插入和编辑内外部链接，以及上传和嵌入图片，支持图片的尺寸调整和对齐方式选择。
7. **富媒体支持**：支持图像、视频、音频等富媒体内容。
8. **插入文件**：支持从本地文件插入各种格式的文件。
9. **表格工具**：提供强大的表格编辑能力，可快速创建和编辑多行多列的表格。
10. **拼写检查**：自动检测并提示拼写错误，保证内容的准确性。
11. **撤销与重做**：提供快捷的撤销和重做功能，便于快速修正误操作。
12. **查找和替换**：通过快捷键或菜单，快速查找和替换文本。

### Markdown 支持
Umo Editor 全面支持 Markdown 语法，用户可以轻松使用 Markdown 语法编写和预览文档。

Markdown 是一种轻量级的标记语言，它使用简单易读的文本格式来描述文档，并使用户能够快速地编写和阅读文档。

### 实用工具集成
Umo Editor 提供了多种实用工具，如二维码、条形码、电子签名、电子签章、流程图、Mermaid 图表、中文大小写转换等。

### 演示模式
Umo Editor 支持演示模式，可以通过屏幕、投影仪等快速将文档分享给团队成员、客户或者参会者，适合于教育培训、工作汇报、会议演示等场景。

### 页面设置
作为一款文档编辑器，Umo Editor 提供了丰富的页面设置选项，包括：页边距设置、页面大小设置、页面方向设置、分页符、换行符、行号、页面水印、页面背景设置等。

### AI 文档助手
Umo Editor 支持 AI 助手，可以根据用户选择的文档内容和输入的指令，调用 AI 模型，智能生成相应的文档内容。

### 文档导出与分享
Umo Editor 支持将文档导出为多种格式的文件，包括 PDF、HTML、图片等格式。

Umo Editor 支持将文档通过链接分享给他人，同时也支持通过 Embed 将文档嵌入到其它 HTML 页面中。

### 气泡菜单与块级菜单
Umo Editor 支持快捷的气泡菜单与块级菜单，可通过快捷菜单快速添加、编辑和删除节点，进一步提升编辑效率。

### 快捷键支持
Umo Editor 提供了丰富的快捷键选项，包括常用快捷键、文字格式快捷键、页面显示快捷键、Markdown 格式快捷键等，用户可以通过快捷键高效地实现文档编辑和操作。

用户可以在 Umo Editor 中通过点击快捷键图标查看所有快捷键列表。

### 支持打印及打印预览
Umo Editor 支持打印预览和打印功能，方便用户查看和打印文档。

### 主题定制
Umo Editor 提供了丰富的主题定制选项，开发者可以通过主题定制轻松定制 Umo Editor 的样式。

### 多语言设置
Umo Editor 支持多语言设置，用户可以根据自己的需求选择中文、英文或其他语言。

### 暗色主题
Umo Editor 支持暗色主题，可以通过设置使用浅色、暗色主题或根据系统设置自动切换。
