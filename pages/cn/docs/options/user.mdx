# 用户配置

用于配置用户相关的数据，包括用户 ID、用户名称、用户头像等。<Badge theme="success">v4.0.0 新增</Badge>

## 默认配置

```js
{
  user: {},
}
```

## 配置项说明

### user.id

**说明**：当前用户 ID。

**类型**：`String`

**默认值**：无

### user.label

**说明**：当前用户名称。

**类型**：`String`

**默认值**：无

### user.avatar

**说明**：当前用户头像。

**类型**：`String`

**默认值**：无
