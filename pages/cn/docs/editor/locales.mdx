# 多语言支持

Umo Editor 内置了对简体中文和英语的支持，如果需要对其他语言进行支持，需要您自行修改和编译代码。

## 官方支持语言列表

* [简体中文 (zh-CN)](https://github.com/umodoc/editor/blob/main/src/locales/zh-CN.json)
* [英语 (en-US)](https://github.com/umodoc/editor/blob/main/src/locales/en-US.json)

## 社区支持语言列表

* [俄语 (ru-RU)](https://github.com/umodoc/editor/blob/main/src/locales/ru-RU.json)：由 [@SerRashin](https://github.com/SerRashin) 提供
* [意大利语 (it-IT)](https://github.com/umodoc/editor/blob/main/src/locales/it-IT.json)：由 [@claudiorivetti](https://github.com/claudiorivetti) 提供
* [藏语 (bo)](https://github.com/umodoc/editor/blob/main/src/locales/bo.json)：由 [@Khangey](https://github.com/Khangey) 提供

## 如何扩展语言

Umo Editor 默认支持的简体中文和英文，如果需要扩展语言，您可以参考简体中文和英文的语言包和配置，自行修改和编译代码。

## 如何贡献语言包

Umo Editor 的发展离不开社区的支持，您可以在 [Umo Editor 仓库](https://github.com/umodoc/editor) 中提交 PR 贡献语言包。
