# 基本介绍

为了保持一致的用户体验，减少开发工作量，自 v3.0 版本开始，Umo Editor 对外提供了一部分的组件。这些组件包括以下内容：

- [UmoMenuButton](./components/menu-button)
- [UmoDialog](./components/dialog)
- [UmoTooltip](./components/tooltip)

这些组件都是基于[TDesign Vue Next](https://tdesign.tencent.com/vue-next/overview)的部分组件开发或二次封装，因此在大部分情况下，它们有着相同的配置和使用方式。
