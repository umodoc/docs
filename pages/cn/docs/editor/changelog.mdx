## 🎉 v10.0.0 <Badge>2026-01-11</Badge>

<Callout type="info" emoji="❗">
  本次更新是一次**里程碑式的大版本升级**。在不破坏原有配置与使用习惯的前提下，Umo
  Editor 底层代码已无痛迁移至 Tiptap v3，并对整体代码进行了系统性重构。
  本次升级不仅显著**降低了打包体积**、**提升了大文档的加载和编辑性能**，也为后续功能扩展和长期维护奠定了更加稳固的基础。**强烈建议所有用户升级。**
</Callout>

---

### 🚀 重磅升级

1. Tiptap 内核从 **v2 升级至 v3 最新版**，全面支持并拥抱 **Tiptap 3**；
2. 生产环境下的打包体积显著降低，整体体积约减少 **3～4 倍**；
3. 大文档加载和编辑性能大幅提升，在相同基准测试下，支持的文档规模约提升 **3 倍以上（纯文本）**；
4. 引入大量全新功能特性，详见下方 **「新增」** 内容；
5. 从根本上修复了一些长期存在的细小问题。

---

### 🔧 重构

1. **全面移除 TypeScript 支持**，核心代码统一使用 JavaScript 重构；
2. 重构了全部 Tiptap 扩展及其相关实现逻辑；
3. 重构工具栏、菜单系统及多处核心交互逻辑，为后续功能扩展提供更清晰的结构；
4. 升级所有依赖到最新版本。
