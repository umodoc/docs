# 外观配置

用于配置外观信息，支持默认外观和现代外观。当同一页面使用了多个 Umo Editor 实例时，以最后一个实例配置为准。<Badge theme="success">v10.0.0 新增</Badge>

## 默认配置

```js
{
  skin: 'default',
}
```

## 配置项说明

### skin

**说明**：默认外观设置。

**类型**：`String`

**默认值**：`default`。

**可选值**：`default` 、 `modern` 。
