## 配置项说明

### dicts.fonts

**说明**：字体列表，用于提供用户可选的字体。

**类型**：`Array`

**配置项**：

- `item.label`: `String` 或 `Object`，字体的显示名称。
- `item.value`: 合法的[CSS font-family](https://developer.mozilla.org/zh-CN/docs/Web/CSS/font-family)值。

### dicts.colors

**说明**：颜色列表，用于提供用户可选的颜色列表，包括字体颜色和背景颜色。

**类型**：`Array`

**配置项**：合法的[CSS 颜色](https://developer.mozilla.org/zh-CN/docs/Web/CSS/color_value)值。

### dicts.lineHeights

**说明**：行高列表，用于提供用户可选的行高列表，包括字体颜色和背景颜色。

**类型**：`Array`

**配置项**：

- `item.label`: `String` 或 `Object`，行高的显示名称。
- `item.value`: 合法的[CSS line-height](https://developer.mozilla.org/zh-CN/docs/Web/CSS/line-height)值。

### dicts.symbols

**说明**：特殊字符列表，用于提供用户可插入的特殊字符列表。 v2.0.0 之前为 `dicts.specialCharacters`

**类型**：`Array`

**配置项**：

- `item.label`: `String` 或 `Object`，特殊字符的分组名称。
- `item.value`: 该分组下所有的特殊字符。

### dicts.emojis

<Callout type="warning" emoji="⚠️">
  **注意**：v6.0.0 版本调整了默认的表情列表。
</Callout>

**说明**：[Emoji](https://baike.baidu.com/item/emoji/8154456)表情列表，用于提供用户可插入的 Emoji 表情列表。

**类型**：`Array`

**配置项**：

- `item.label`: `String` 或 `Object`，Emoji 的分组名称。
- `item.value`: 该分组下所有的 Emoji 表情，需使用空格分隔开。

**数据库支持**：Emoji 本质上是 Unicode 编码，长度为 4 个字节，需要数据库支持存储，[如何让MySql数据库支持emoji表情](https://juejin.cn/post/7106906047670910989)，您也可以通过以下配置来禁用通过菜单插入 Emoji，但这并不能防止用户通过粘贴、输入法等方式插入 Emoji。

<Tabs.Tab>
```js
{
  toolbar:{
    disableMenuItems: ['emoji']
  }
}
```
</Tabs.Tab>

### dicts.pageSizes

**说明**：页面大小列表，用于提供可选择的页面大小列表和默认页面大小。

**类型**：`Array`

**配置项**：

- `item.label`: `String` 或 `Object`，Emoji 的分组名称。
- `item.width`: 页面宽度，单位为厘米(cm)。
- `item.height`: 页面高度，单位为厘米(cm)。
- `item.default`: 是否为默认页面大小。
