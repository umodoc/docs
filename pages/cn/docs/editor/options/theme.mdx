# 主题配置

用于配置主题信息，支持浅色主题、暗色主题和跟随系统设置。更多主题配置请参考[主题定制](../themes)。

当同一页面使用了多个 Umo Editor 实例时，以最后一个实例配置为准。

## 默认配置

```js
{
  theme: 'light',
}
```

## 配置项说明

### theme

**说明**：默认主题。

**类型**：`String`

**默认值**：`light`。

**可选值**：`light` 、 `dark` 、 `auto`。
