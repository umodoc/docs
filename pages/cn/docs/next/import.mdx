# 导入能力

此能力允许将外部数据导入到 Umo Editor 编辑器中。提供多种形式的导入。

# 导入Markdown文本 
Markdown 是一种轻量级标记语言，作为通用的文本格式，被绝大多数富文本编辑器支持。用户只需输入标准的 Markdown 语法，编辑器即可将其渲染为对应的富文本格式，并插入到当前光标所在位置。

## Markdown扩展

### 高亮块
使用`：：：`来标识高亮块的开始和结束。
```
:::info
高亮块
:::
```

## 方法列表

### insertMarkdown

**说明**：插入markdwon格式文本。

**参数**：`markData`，String，Markdown文本。

**参数**：`exRules`，object，Markdown扩展规则。

**返回值**：无
