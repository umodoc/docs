# 核心特性

以下是 Umo Editor Next 即将支持或已经支持的特性列表，目前的完成进度请查看[开发计划](./roadmap)。该列表可能会不定期更新，以添加更多的功能特性。

## 多人协作编辑功能

- 支持多人实时协作编辑文档，提供冲突检测和版本合并功能； 
- 用户可以实时查看他人光标位置和修改内容，避免冲突； 
- 支持协作模式下的文档批注/评论功能，方便协作成员实时讨论。

## 更完整的 Word 导入和导出功能

- 支持复杂的 Word 文档格式，确保导入和导出时格式及样式不丢失；  
- 支持嵌套表格、复杂样式、目录大纲等高级功能，导入后自动优化排版，导出时保留原始样式。  

## PDF 静默导出功能

- 一键静默导出 PDF，无需弹窗或额外操作，提升效率和体验；  
- 支持添加水印、密码保护等功能；
- 导出的 PDF 文件还原度高、兼容性强，可直接用于打印或分享。

## 更强大和友好的表格功能

- 表格样式可灵活自定义，包括边框、主题、单元格样式、背景、分页等，满足复杂的表格编辑需求；  
- 支持列宽、列高、表格大小自定义，可灵活拖动调整大小；  
- 支持粘贴 Excel、Word 等格式，方便与其他工具协作。

## 嵌入式 AI 文档助手

- 通过侧边栏形式提供 AI 助手，支持智能补全、内容生成和问题解答；
- AI 助手可根据上下文提供写作建议，自动生成摘要，帮助用户校对语法和格式；
- 还支持根据用户需求生成图表、表格或其他内容，提升文档创作效率。

## 文档历史版本管理功能

- 显示文档的历史版本，支持版本回溯和对比；
- 用户可以查看每个版本的修改记录，标记重要版本，并恢复到任意历史版本。

## 支持 Web 版式

- 支持将文档导出为适配 Web 的格式，便于在线浏览和分享；
- 提供响应式设计，确保在不同设备上都有良好的阅读体验。
