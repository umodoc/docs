# 插入图标

在 Umo Editor Next 中，您可以方便快捷的插入各类图标，图标库默认采用 [IconPark](https://iconpark.oceanengine.com/official)，提供了 2000 多个图标和自定义选项，您也可以自定义图标或者扩展图标。

## 效果截图

![Umo Editor Icons](/demos/icons-cn.png)

## 图标相关配置

```js
const defaultOptions = {
  // 页边栏相关配置
  icons: {
    defaultLineWidth: 4
  },
}
```

## 配置项说明

### icons.defaultLineWidth

**说明**：图标默认的默认线条宽度。

**类型**：`Number`
