# 更新日志

## 🚀 v13.0.1

同步 Umo Editor v11.0.3 的最新更改。

## 🎉 v13.0.0

本次更新同步了 Umo Editor v11.0.0 的最新更改，重点增强了表格能力、导入导出体验和大文档性能，并修复了协作、评论和菜单相关问题。详细更新日志见：[Umo Editor 更新日志](../editor/changelog)。

### 📊 表格

1. 修正表格左侧行手柄与顶部手柄和行高、列宽的对齐问题；
2. 表格在演示模式下隐藏行高、列宽调整手柄；
3. 修复表格高度自适应异常，以及高度被行高拖拽意外固定的问题；
4. 修复表格行高拖动手柄失效的问题；
5. 优化表格行高、列宽拖动手柄在只读/编辑状态下的显隐；
6. 超宽表格支持通过滚动条查看；
7. 优化表格在大文档、多表格场景下的性能和交互；
8. 表格创建时自动将焦点设置到第一个单元格中；
9. 修复增加表格列时，列手柄数量未同步更新的问题；
10. 大幅优化表格渲染性能，提升大文档加载速度。

### 📥 导入/导出

1. 优化不同文档结构下的 Word 导入效果；
2. Word 导出支持导出图片描述信息；
3. 优化 Office 文档导入时行高可能不正确的问题；
4. 添加 Office 粘贴大内容提示；
5. 修复批注对内联节点的错误块包装；
6. 修复 Word 导出时由于颜色转换错误引起的报错等问题；
7. 修复 Excel 粘贴表格可能出现异常的问题；
8. 优化图片导出，改为直接从 DOM 中获取渲染后的图片，而不是远程加载图片，以提升导出速度和稳定性；
9. 优化 Markdown 表格导入功能，支持单元格对齐方式。

### ⚡ 性能优化

1. 大幅优化大文档的渲染性能，提升大文档加载速度。

### 🧩 其他修复与调整

1. 修复协作参数通过 URL 透传失效的问题；
2. 默认关闭协同离线存储，可通过 [`collaboration.provider.enableIndexedDB`](./collaboration#) 配置开启；
3. 修复菜单项可能偶发性不动态更新的问题；
4. 修复评论关闭后保存可能会报错的问题；
5. 修复 sticky tool 菜单可能跑到编辑器容器外的问题；
6. 其他细节优化和调整。

## 🚀 v11.3.0

本次更新重点增强了 Word 导出的稳定性、可观测性和内容还原能力，图片、表格、书签、页码、评论、修订等常见场景的导出效果更完整，程序化调用也更适合集成。

1. 导出稳定性增强，失败时更容易降级而不是整次报错
2. 导出结果提示增强，支持成功、失败、存在降级三种状态
3. 组件 `exportDocx()` 新增 `onDiagnostics` 和 `onExportResult`
4. 图片导出增强，支持浮动、环绕、旋转、翻转和失败降级
5. 图片替代文本增强，支持导出 `alt`、`title`、`name`
6. 表格导出增强，支持更多边框、条纹底色、行高和列宽处理
7. 表格合并单元格导出稳定性增强
8. 书签和内部跳转导出增强
9. 页码和字段导出增强，支持总页数和起始页码
10. 页边框支持通过 `docxOptions.page.borders` 配置
11. 文档元数据支持通过 `docxOptions.documentMeta` 配置
12. 评论线程和评论回复导出增强
13. 文本级和段落级修订导出增强
14. 默认字号、列表位置、代码块样式等细节做了优化
15. 浮动图片、表格、分栏等复杂场景持续修复回归问题
16. 文件、音频、视频、网页目前导出为链接占位，不是原生嵌入对象
17. 图片、图标、图表导出仍受浏览器环境影响
18. 复杂非文本修订仍以保内容为主，不保证完整保留原生修订外形
