# 文字提示组件

文字提示组件用来显示文字提示信息。

## 示例

见：https://demo.umodoc.com/components?lang=zh-CN

## 使用方法

```vue
<template>
  <umo-tooltip content="提示文字">
    <button>按钮</button>
  </umo-tooltip>
</template>

<script setup>
import { UmoTooltip } from '@umoteam/editor'
</script>
```


## 属性（Props）

### content

**说明**：要提示的文字内容

**类型**：无
