# 相关用户配置

用于配置与文档相关的用户列表。该配置通常与提及他人、多人协作编辑、文档评论、自定义扩展或业务节点一起使用。

## 默认配置

```js
{
  users: [],
}
```

## 配置项说明

### users

**说明**：相关用户列表。

**类型**：`Array`

**默认值**：`[]`

列表中的每一项通常可以包含：

- `id`：用户 ID
- `label`：用户名称
- `avatar`：头像地址
- `color`：用户颜色，用于快速区分不同用户
- `name` / `nickname`：业务自定义名称字段，可按需扩展

## 使用示例

```js
const options = {
  users: [
    { id: 'u_10001', label: '张三' },
    { id: 'u_10002', label: '李四' },
    { id: 'u_10003', label: '王五' },
  ],
}
```

## 使用建议

- 如果业务使用了提及、人员选择、自定义审批节点等功能，建议预先传入用户列表。
- 如果用户量较大，建议在业务层做搜索、分页或懒加载，而不是一次性传入全部数据。
