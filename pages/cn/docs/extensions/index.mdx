# 基础介绍

Umo Editor 基于 [Tiptap](https://tiptap.dev/) 无头编辑器开发，支持通过 [官方插件](https://tiptap.dev/docs/editor/extensions)、[第三方插件](https://www.npmjs.com/search?q=tiptap%20extension) 和 [自定义插件](https://tiptap.dev/docs/editor/guide/custom-extensions) 来增强功能和用户体验。


## UI 相关说明

Umo Editor 目前只开放了部分 UI 相关的功能，如 [弹层组件](../docs/popup)，所使用的插件涉及的 UI 部分目前需要自行实现，为了保持 UI 的一致性，您可以使用 Umo Editor 提供的 [CSS Variables](../docs/theme#css-variables)，其他 UI 相关的内容将在后续版本中逐步开放。您也可以拉取代码自行开发或 [联系我们](../docs#联系我们) 进行 [定制开发](../docs#定制开发)。
