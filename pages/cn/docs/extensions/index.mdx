# 基础介绍

Umo Editor 基于[Tiptap](https://tiptap.dev/)无头编辑器开发，支持通过[官方扩展](https://tiptap.dev/docs/editor/extensions)、[第三方扩展](https://www.npmjs.com/search?q=tiptap%20extension)和[自定义扩展](https://tiptap.dev/docs/editor/guide/custom-extensions)来增强功能和用户体验。


## UI 相关说明

Umo Editor 在 v3.0 开始，开放了部分 UI 相关的组件，见[组件列表](./components)。除已开放的组件外，为了保持 UI 的一致性，您可以在您的 UI 中使用 Umo Editor 提供的[CSS 变量](../docs/themes#css-variables)。您也可以拉取代码自行开发或[联系我们](../docs#联系我们)进行[定制开发](../docs#定制开发)。

## 内置扩展列表

见[内置扩展列表](./extensions/built-in)。
