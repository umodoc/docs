<FAQBox title="我已经正确安装并配置了 Umo Editor，但是无法显示。">
请检查以下几点：

1. `<umo-editor />` 的父节点是否有高度，或者有样式 `overflow: hidden`。
2. 检查浏览器是否支持，详见 [浏览器支持](/docs/#浏览器支持)。
3. 右键查看元素，检查是否加载了 Umo Editor 的 DOM 节点。
4. 检查浏览器的 Console 面板，是否有报错信息。
</FAQBox>
