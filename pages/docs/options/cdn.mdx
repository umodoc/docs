# CDN 配置

用于配置 CDN 路径。为了减小 Umo Editor 核心包的体积和提升加载速度，Umo Editor 使用 CDN 引入部分静态资源，这些静态资源包含第三方 JS 插件、图片、图标资源等。

## 默认配置

```js
{
  cdnUrl: 'https://unpkg.com/@umodoc/editor@latest/dist/umd',
}
```

## 使用公共 CDN

```js
{
  // 使用 unpkg.com
  cdnUrl: 'https://unpkg.com/@umodoc/editor@latest/dist/umd',
  // 或使用 jsdelivr.net
  // cdnUrl: 'https://cdn.jsdelivr.net/npm/@umodoc/editor@latest/dist/umd',
}
```

## 使用本地资源

对于内网环境，或者对想进一步提高加载速度等场景，您可以下载对应版本的静态资源后，解压放置到合适的位置，并正确的配置 `cdnUrl`，注意不要以 `/` 结尾。

- [Umo Editor v1.0.x - CDN 资源下载](https://cdn.umodoc.com/cdn%2FUmo%20Editor%20v1.0.x%20-%20CDN%20%E8%B5%84%E6%BA%90.zip)；

## 配置项说明

### cdnUrl

**说明**：CDN 路径。

**类型**：`String`

**默认值**：`https://unpkg.com/@umodoc/editor@latest/dist/umd`

**配置项**：合法的 [URL](https://developer.mozilla.org/zh-CN/docs/Learn/Common_questions/Web_mechanics/What_is_a_URL) 值，不要以 `/` 结尾。
