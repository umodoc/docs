# User Configuration

Used to configure user-related data, including user ID, user name, user avatar, etc.<Badge theme="success">Added in v4.0.0</Badge>

## Default Configuration

```js
{
  user: {},
}
```

## Configuration Description

### user.id

**Description**: The current user ID.

**Type**: `String`

**Default Value**: None

### user.label

**Description**: The current user name.

**Type**: `String`

**Default Value**: None

### user.avatar

**Description**: The current user avatar.

**Type**: `String`

**Default Value**: None
