# Related Users Configuration

Used to configure data for related users, including user IDs, user names, etc., for scenarios such as "mentioning users (i.e., @ users)". <Badge theme="success">Added in v6.0.0</Badge>

## Default Configuration

```js
{
  users: [],
}
```

## Configuration Description

**Description**: Data for related users.

**Type**: `Array`

**Default Value**: `[]`

**Configuration Items**:

- `users[].id`: `String`, user ID.
- `users[].label`: `String`, user name.
