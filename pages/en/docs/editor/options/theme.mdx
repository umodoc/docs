# Theme Configuration

Used to configure theme information, supporting light themes, dark themes, and following the system settings. For more theme configuration, please refer to [Theme Customization](../themes).

When multiple Umo Editor instances are used on the same page, the configuration of the last instance takes precedence.

## Default Configuration

```js
{
  theme: 'light',
}
```

## Configuration Description

### theme

**Description**: Default theme.

**Type**: `String`

**Default Value**: `light`

**Optional Values**: `light`, `dark`, `auto`
