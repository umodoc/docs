# Insert Icons

In Umo Editor Next, you can easily insert a wide variety of icons. By default, the icon library uses [IconPark](https://iconpark.oceanengine.com/official), which offers over 2,000 icons and customization options. You can also add your own icons or extend the icon set as needed.

![Umo Editor Icons](/demos/icons-en.png)

## Icon Configuration

```js
const defaultOptions = {
  // Sidebar related configuration
  icons: {
    defaultLineWidth: 4
  },
}
```

## Configuration Details

### icons.defaultLineWidth

**Description**: The default line width for icons.

**Type**: `Number`
