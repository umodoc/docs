# Development Roadmap

Umo Editor Next is a commercial software. We will gradually add the following features or functionalities to this software:

- [x] Multi-user collaboration editing feature
- [ ] More comprehensive Word import and export functionality
- [ ] Silent PDF export functionality
- [ ] Document annotation/commenting functionality
- [ ] More powerful and user-friendly table features
- [ ] Embedded AI document assistant (in the form of a sidebar IM)
- [ ] Document version history management functionality
- [ ] Web layout support

If you have better suggestions, you are also welcome to [contact us](../#contact-us).
