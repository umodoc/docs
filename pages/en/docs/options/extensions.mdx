# Custom Extension Configuration

This section is used to configure custom extensions. For more information on custom extensions, please refer to the [Custom Extensions](/en/docs/extensions) section.

## Default Configuration

```js
{
  extensions: [],
}
```

## Configuration Item Description

### extensions

**Description**: A list of custom extensions.

**Type**: `Array`

**Default Value**: `[]`.

**Options**: Valid Tiptap extensions.
