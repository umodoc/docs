# Theme Configuration

Configures theme settings, supporting light themes, dark themes, and system preference adherence.

## Default Configuration
```js
{
  theme: 'light',
}
```

## Configuration Item Description

### theme

**Description**：The default theme setting.

**Type**：`String`

**Default**：`light`。

**Options**：`light` 、 `dark` 、 `auto`。
