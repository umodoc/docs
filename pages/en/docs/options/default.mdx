<Callout type="error">
Starting from version v2.0.0, `dicts` under `label` and `document.placeholder` support both `String` and `Object` types. The `Object` type is mainly used for multi-language configuration information, formatted as `{ en_US: 'English', zh_CN: 'Chinese' }`.
</Callout>

## Configuration Sections

The configuration items of Umo Editor are mainly divided into the following parts:

- `editorKey`: String, the key for the editor, used to distinguish different editor instances.
- `locale`: String, the display language, defaults to Chinese.
- `height`: String, the height of the editor.
- `dicts`: Object, dictionary configuration, see [Dictionaries Configuration](./dicts).
- `toolbar`: Object, toolbar configuration, see [Toolbar Configuration](./toolbar).
- `page`: Object, page configuration, see [Page Configuration](./page).
- `document`: Object, document configuration, see [Document Configuration](./document).
- `assistant`: Object, configuration for the AI Assistant, see [AI Assistant](./assistant) for details. <Badge theme="success">New in v3.0.0</Badge>
- `templates`: Array, preset page template configurations, see [Template Configuration](./templates).
- `cdnUrl`: String, the base path for static resources, must be a valid URL, see [CDN Configuration](./cdn).
- `shareUrl`: String, share link configuration, must be a valid URL, see [Share Configuration](./share).
- `diagrams`: Object, Diagrams chart configuration, see [Diagrams Chart Configuration](./diagrams).
- `file`: Object, file upload configuration, see [File Upload Configuration](./file).
- `extensions`: Array see [Custom Extensions](./extensions). <Badge theme="success">New in v1.4.0</Badge>
- `translations`: Object see [Custom Localization Configuration](./locales). <Badge theme="success">New in v2.0.0</Badge>
- `onSave(content, page, document)`: AsyncFunction, document save method, should return a Promise, see [Methods](/en/options/methods#onsave).
- `onFileUpload(file)`: AsyncFunction, file upload method, should return a Promise, see [Methods](/en/docs/options/methods#onfileupload).
- `onFileDelete(id, url)`: Function, file delete method, see [Methods](/en/docs/options/methods#onfiledelete).
- `onAssistant(payload, content)`: AsyncFunction, the method for handling data returned by the AI Assistant, see [AI Assistant](/cn/docs/options/methods#onassistant) for details. <Badge theme="success">New in v3.0.0</Badge>
- `onCustomImportWordMethod(file)`: Async Function, a custom method for importing Word files. See [Method Configuration](/en/docs/options/methods#oncustomimportwordmethod) for more details. <Badge theme="success">New in v3.1.0</Badge>
