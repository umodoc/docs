## Configuration Description

The configuration options of Umo Editor are mainly divided into the following parts:

- `editorKey`: String, the editor key, used to distinguish different editor instances.
- `locale`: String, display language, default is Chinese. <Badge theme="success">Added in v2.0.0</Badge>
- `theme`: String, theme setting, default is light theme. <Badge theme="success">Added in v2.1.0</Badge>
- `height`: String, editor height.
- `fullscreenZIndex`: Number, the z-index of the container in fullscreen mode, used to prevent being covered by other elements, such as when used in a popup layer. <Badge theme="success">Added in v6.0.0</Badge>
- `dicts`: Object, dictionary configuration, see [Dictionary Configuration](./dicts).
- `toolbar`: Object, toolbar configuration, see [Toolbar Configuration](./toolbar).
- `page`: Object, page configuration, see [Page Configuration](./page).
- `document`: Object, document configuration, see [Document Configuration](./document).
- `ai`: Object, AI document assistant configuration, see [AI Document Assistant](./ai). <Badge theme="success">Added in v3.0.0</Badge>
- `echarts`: Object, ECharts chart configuration, see [ECharts Configuration](./echarts). <Badge theme="success">Added in v6.0.0</Badge>
- `webPages`: Object, web page configuration, see [Web Pages](./webpages). <Badge theme="success">Added in v6.0.0</Badge>
- `templates`: Array, preset page template configuration, see [Template Configuration](./templates).
- `cdnUrl`: String, base path for static resources, must be a valid URL, see [CDN Configuration](./cdn).
- `shareUrl`: String, share link configuration, must be a valid URL, see [Share Configuration](./share).
- `diagrams`: Object, Diagrams chart configuration, see [Diagrams Chart Configuration](./diagrams).
- `user`: Object, current user configuration, see [User Configuration](./user). <Badge theme="success">Added in v4.0.0</Badge>
- `users`: Array, related user configuration, see [Related User Configuration](./users). <Badge theme="success">Added in v6.0.0</Badge>
- `file`: Object, file upload configuration, see [File Upload Configuration](./file).
- `extensions`: Array, see [Custom Extensions](./extensions). <Badge theme="success">Added in v1.4.0</Badge>
- `translations`: Object, see [Custom Translation Information](./locales). <Badge theme="success">Added in v2.0.0</Badge>
- `onSave(content, page, document)`: Async Function, document save method, should return a Promise, see [Method Configuration](./methods#onsave).
- `onFileUpload(file)`: Async Function, file upload method, should return a Promise, see [Method Configuration](./methods#onfileupload).
- `onFileDelete(id, url)`: Function, file delete method, see [Method Configuration](./methods#onfiledelete).
- `onCustomEChartSettings(data, config)`: Function, custom ECharts chart configuration method, see [Method Configuration](./methods#oncustomcehartsettings). <Badge theme="success">Added in v6.0.0</Badge>
- `onCustomImportWordMethod(file)`: Async Function, custom Word file import method, see [Method Configuration](./methods#oncustomimportwordmethod). <Badge theme="success">Added in v3.1.0</Badge>
