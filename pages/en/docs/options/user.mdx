# User Configuration

Used to configure user-related data, including user ID, user name, user avatar, etc.

## Default Configuration

```js
{
  user: {},
}
```

## Configuration Options

### user.id

**Description**: The current user ID.

**Type**: `String`

**Default Value**: None

### user.label

**Description**: The current user name.

**Type**: `String`

**Default Value**: None

### user.avatar

**Description**: The current user avatar.

**Type**: `String`

**Default Value**: None
