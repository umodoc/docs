## Configuration Description

### assistant.enabled

**Description**: Whether to enable the AI Document Assistant feature.

**Type**: `Boolean`

**Default Value**: `false`

### assistant.maxlength

**Description**: The maximum length of commands that the AI Document Assistant can process.

**Type**: `Number`

**Default Value**: `100`

### assistant.commands

**Description**: Commonly used command configurations in the AI Document Assistant panel.

**Type**: `Array`

**Default Value**:

[
  {
    label: { en_US: 'Continuation', zh_CN: '续写', ru_RU: 'Продолжение' },

**Configuration Items**:

- `item.label`: `String` or `Object`, the text displayed in the AI Document Assistant panel.
- `item.value`: `String` or `Object`, the actual command sent to `onAssistant`. See [AI Document Assistant Configuration](../methods#onAssistant).
- `item.autoSend`: `Boolean`, whether to automatically send the command to `onAssistant` when the user selects it. Defaults to `true`. If set to `false`, the user can modify the command and manually click the button to send it.
