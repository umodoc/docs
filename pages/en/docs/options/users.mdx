# Related Users Configuration

Used to configure data for related users, including user IDs, user names, etc., for scenarios such as "mentioning users (i.e., @ users)".

## Default Configuration

```js
{
  users: [],
}
```

## Configuration Options

**Description**: Data for related users.

**Type**: `Array`

**Default Value**: `[]`

**Configuration Items**:

- `users[].id`: `String`, user ID.
- `users[].label`: `String`, user name.
