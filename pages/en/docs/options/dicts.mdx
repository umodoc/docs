## Configuration Description

### dicts.fonts

**Description**: Font list, used to provide selectable fonts for users.

**Type**: `Array`

**Options**:

- `item.label`: `String` or `Object`, the display name of the font.
- `item.value`: A valid [CSS font-family](https://developer.mozilla.org/en-US/docs/Web/CSS/font-family) value.

### dicts.colors

**Description**: Color list, used to provide selectable colors for users, including font colors and background colors.

**Type**: `Array`

**Options**: Valid [CSS color](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value) values.

### dicts.lineHeights

**Description**: Line height list, used to provide selectable line heights for users.

**Type**: `Array`

**Options**:

- `item.label`: `String` or `Object`, the display name of the line height.
- `item.value`: A valid [CSS line-height](https://developer.mozilla.org/en-US/docs/Web/CSS/line-height) value.

### dicts.symbols

**Description**: Special character list, used to provide selectable special characters for users. Before v2.0.0, it was `dicts.specialCharacters`.

**Type**: `Array`

**Options**:

- `item.label`: `String` or `Object`, the group name of the special characters.
- `item.value`: All special characters under this group.

### dicts.emojis

**Description**: [Emoji](https://en.wikipedia.org/wiki/Emoji) list, used to provide selectable emojis for users.

**Type**: `Array`

**Options**:

- `item.label`: `String` or `Object`, the group name of the emojis.
- `item.value`: All emojis under this group, separated by spaces.

**Database Support**: Emojis are essentially Unicode characters with a length of 4 bytes, requiring database support for storage. [How to enable MySQL database support for emojis](https://juejin.cn/post/7106906047670910989). You can also disable emoji insertion through the menu with the following configuration, but this does not prevent users from inserting emojis via copy-paste or input methods.

    disableMenuItems: ['emoji']

### dicts.pageSizes

**Description**: Page size list, used to provide selectable page sizes and default page size.

**Type**: `Array`

**Options**:

- `item.label`: `String` or `Object`, the name of the page size.
- `item.width`: Page width in centimeters (cm).
- `item.height`: Page height in centimeters (cm).
- `item.default`: Whether it is the default page size.
