## Configuration Item Descriptions

### dicts.fonts

**Description**: A list of fonts that provides users with selectable font options.

**Type**: `Array`

**Configuration Items**:

- `label`: `String` or `Object`, the display name of the font.
- `value`: A valid [CSS font-family](https://developer.mozilla.org/en-US/docs/Web/CSS/font-family) value.

### dicts.colors

**Description**: A list of colors that provides users with a selection of font and background colors.

**Type**: `Array`

**Configuration Items**: Valid [CSS color](https://developer.mozilla.org/en-US/docs/Web/CSS/color_value) values.

### dicts.lineHeights

**Description**: A list of line heights that provides users with selectable line height options.

**Type**: `Array`

**Configuration Items**:

- `label`: `String` or `Object`, the display name for the line height.
- `value`: A valid [CSS line-height](https://developer.mozilla.org/en-US/docs/Web/CSS/line-height) value.

### dicts.symbols

**Description**: A list of special characters that provides users with a selection of special characters to insert. Prior to v2.0.0, this was known as `dicts.specialCharacters`.

**Type**: `Array`

**Configuration Items**:

- `label`: `String` or `Object`, the group name for the special characters.
- `items`: All special characters under this group.

### dicts.emojis

**Description**: A list of emoji that provides users with a selection of emoji to insert.

**Type**: `Array`

**Configuration Items**:

- `label`: `String` or `Object`, the group name for the Emoji.
- `items`: All Emoji under this group, separated by spaces.

**Database Support**: Emojis are essentially Unicode characters, 4 bytes in length, and require database support for storage. You can disable the insertion of Emojis through the menu with the following configuration, but this will not prevent users from inserting Emojis through pasting or input methods.

<Tabs items={['Global Configuration', 'SFC Configuration']}>
<Tabs.Tab>
```js
{
  toolbar:{
    disableMenuItems: ['emoji']
  }
}
```
</Tabs.Tab>

</Tabs.Tab>
</Tabs>

### dicts.pageSizes

**Description**: A list of page sizes that provides selectable page sizes and a default page size.

**Type**: `Array`

**Configuration Items**:

- `label`: `String` or `Object`, the display name for the page size.
- `width`: The width of the page in centimeters (cm).
- `height`: The height of the page in centimeters (cm).
- `default`: Whether it is the default page size.
