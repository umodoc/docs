# Introduction

Umo Office Convert converts Microsoft Office documents, WPS Office documents, OpenDocument (ODF), Unified Office Format (UOF), and other common office formats into formats that can be rendered by [Umo Office Viewer](./office-viewer).

This software is commercial and requires a purchase before use. When using Umo Office Convert, you must comply with the [Umo Team Commercial Software License Agreement](./license).

## Supported File Formats

Umo Office Convert supports 40+ formats, including:

- Microsoft Office: Word/Excel/PowerPoint (DOC/DOCX, XLS/XLSX, PPT/PPTX, and more)
- WPS Office: WPS/ET/DPS and templates
- ODF: ODT/ODS/ODP/ODG and related templates
- UOF: UOT/UOS/UOP
- Common formats: PDF, RTF, CSV

## Quickstart

### 1. Prerequisites

See the `README.md` in the project repository.

### 2. Configure environment variables

See [Environment Variables](./office-convert/env).

### 3. Install dependencies

```bash npm2yarn
npm install
```

### 4. Start the service

```bash npm2yarn
npm run start
```

### 5. Open API docs

After the service starts, visit `http://localhost:[PORT]/openapi` to view the Swagger documentation.
