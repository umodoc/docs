# Environment Variables

Umo Office Convert uses a `.env` file to configure runtime settings. Copy `.env.example` to `.env` in the project root and adjust values for your environment.

## Node

```bash
NODE_ENV = production
```

## Service

```bash
PORT = 1236
SERVER_NAME = Umo Office Convert
```

## Logging

```bash
LOG_DIR = ./logs
```

## Upload & Storage

```bash
MAX_FILE_SIZE = 50
UPLOAD_DIR = ./uploads/[year]/[month]
CONVERTED_DIR = ./converted/[year]/[month]
FILE_HASH_ALGORITHM = sha256
```

## MySQL

```bash
DB_HOST = localhost
DB_PORT = 3306
DB_USER = root
DB_PASSWORD =
DB_NAME = umo_office_convert
DB_TABLE_CONVERT_TASKS = convert_tasks
```

## Webhook

```bash
CONVERTED_WEBHOOK_URL =
```

## Access Control

```bash
WHITE_LIST = 127.0.0.1, 0.0.0.0, localhost
```
