# Tooltip Component

The tooltip component is used to display text hints.

## Example

See: https://demo.umodoc.com/components?lang=en-US

## Usage

```vue
<template>
  <umo-tooltip content="Hint text">
    <button>Button</button>
  </umo-tooltip>
</template>

<script setup>
import { UmoTooltip } from '@umoteam/editor'
</script>
```

## Props

### content

**Description**：The text content to be displayed as a hint.

**Type**：None
