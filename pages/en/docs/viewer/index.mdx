# Introduction

> Note: Starting from version v1.0.0, Umo Viewer no longer supports PDF document preview.

Umo Viewer is a document viewer for previewing Umo Editor documents. It was created to solve the preview problem of [Umo Editor](https://github.com/umodoc/editor) documents. In most non-editing scenarios, only the document content needs to be displayed without loading the entire editor to improve loading speed. Umo Viewer can seamlessly integrate with [Umo Editor](https://github.com/umodoc/editor) documents and maintain good compatibility.

[Umo Editor](https://github.com/umodoc/editor) is a localized open-source document editor based on Vue3 and Tiptap3. It provides powerful document editing capabilities, AI creation features, and supports pagination, Markdown syntax, rich text editing, and much more.

## Key Features

- Support for previewing [Umo Editor](https://github.com/umodoc/editor) document content
- Password protection for documents
- Extensive customization options
- Multi-language support
- Custom themes
- Custom document metadata
- Print and share
- Automatic outline extraction
- Zoom and responsive layout
- Display in modal dialogs or embed via iframe

## Products from Umo Team

- [Umo Editor](https://dev.umodoc.com/en/docs/editor): A localized open-source document editor based on Vue3 and Tiptap.
- [Umo Viewer](https://dev.umodoc.com/en/docs/viewer): An open-source, lightweight document viewer for Umo Editor.
- [Umo Editor Next](https://dev.umodoc.com/en/docs/next): An enhanced version of Umo Editor that includes all features of the latest Umo Editor, with enhanced support for multi-user real-time collaboration, document annotations (comments), version history, AI-assisted creation, import/export, tables, and more. It also improves the customization of toolbars and sidebars.
- [Umo Editor Server](https://dev.umodoc.com/en/docs/server): A companion server-side software developed to extend Umo Editor capabilities such as multi-user collaborative editing, document annotations, and document import/export.
- [Umo Office Viewer](https://dev.umodoc.com/en/docs/office-viewer): An Office document viewer that supports previewing over 40 mainstream office document formats in web pages.
- [Umo Office Convert](https://dev.umodoc.com/en/docs/office-convert): Converts over 40 office formats into web-viewable formats, ready to pair with Umo Office Viewer for seamless online document preview.
