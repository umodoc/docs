# Getting Started

This guide helps you integrate `umo-office-viewer` into a Vue 3 project.

## 1. Install

### 1) Clone the private repository

```bash
git clone <Umo Office Viewer private repository URL>
```

### 2) Install dependencies and build

```sh npm2yarn
npm install
npm run build
```

### 3) Install the package

Assume the local repo is located at `../umo-office-viewer` relative to your application:

```sh npm2yarn
npm install --save ../umo-office-viewer/releases/umo-office-viewer-xxx.tgz
```

## 2. Usage

### 1) Global install

```js filename="main.js"
import { createApp } from 'vue'
import { useUmoOfficeViewer } from '@umoteam/office-viewer'

const app = createApp(App)
app.use(useUmoOfficeViewer, {})
app.mount('#app')
```

### 2) Component usage

```vue filename="Component.vue"
<template>
  <umo-office-viewer v-bind="options" />
</template>

<script setup>
import { ref } from 'vue'
import { UmoOfficeViewer } from '@umoteam/office-viewer'

const options = ref({})
</script>
```

### 3) Nuxt 3

Umo Office Viewer does not support SSR. Use `<client-only>` to render it on the client side.

```vue filename="Component.vue"
<template>
  <client-only>
    <umo-office-viewer v-bind="options" />
  </client-only>
</template>
```

## 3. Configuration

See [Options](./options). Before configuring, it is recommended to run [Umo Office Convert](../office-convert/deployment).
