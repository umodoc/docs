# Features

Umo Office Viewer focuses on online preview of Office documents and PDFs, aiming to be simple to integrate, extensible, and suitable for private (intranet) deployment.

## Overview

- Office/PDF preview component
- Theme modes: `light` / `dark` / `auto` with CSS variables for branding
- Internationalization: `zh-CN` / `en-US` (extensible)
- Full set of events and methods for integration
- Intranet-friendly deployment with no external dependencies

## Capabilities

### Preview

- Office and PDF preview
- Single-page or multi-page layout
- Fit-width display

### Controls

- Fullscreen, rotate, zoom, jump to page
- Print, download original, export PDF, share link
- Sidebar panels (info, thumbnails)

### Security

- Watermark (text, grid, opacity, rotation, font size)
- Password validation for protected documents
