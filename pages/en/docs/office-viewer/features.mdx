# Features

Umo Office Viewer focuses on online preview of Office documents and PDFs, aiming to be simple to integrate, extensible, and suitable for private (intranet) deployment.

## Overview

- Professional preview component for Office documents and PDFs
- Theme modes (`light`/`dark`/`auto`) with rich CSS variables for branding
- Internationalization and locale switching (`zh-CN`/`en-US`)
- Complete events and methods for integrating with business logic
- Intranet deployable with zero external network dependencies

## Core Capabilities

### Preview

- Online preview for Office documents and PDFs
- Single-page / multi-column (multi-page) layout switching, with fit-width display

### Controls

- Fullscreen preview, page rotation, zoom, jump to a specific page
- Print preview & printing, download original document, export PDF, document sharing
- Sidebar info panel and thumbnail navigation

### Security & Compliance

- Page watermark (text, grid, color, rotation, font size, etc.)
- Password validation for protected documents
- Suitable for enterprise intranet environments

### Internationalization & Theme

- Built-in Chinese and English locales, and you can extend more locales as needed
- Override CSS variables to customize branding

## Intranet Deployment

- Intranet-friendly: no reliance on external CDNs or online services
- Ready to use: import the plugin and pass required options; see [Getting Started](./getting-started)

## Recommended Scenarios

- Enterprise document preview, contract approval, archive management, knowledge base browsing
- Teaching courseware preview, technical documentation and report distribution
- Any web scenario that needs Office/PDF preview and printing
