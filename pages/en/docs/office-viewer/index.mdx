# Introduction

Umo Office Viewer is an Office document viewer that supports viewing mainstream Microsoft Office documents, WPS Office documents, OpenDocument Format (ODF), Unified Office Format (UOF), and other common office formats in web pages, and also supports PDF preview. For the full list, see [Supported File Formats](#supported-file-formats). Built with Vue 3, it is designed to be lightweight and easy to integrate for online document preview.

This plugin is commercial and requires a license purchase before use. After purchasing the commercial source edition, you will receive the full source code and access to the private repository, which means you can always obtain the latest code and future feature updates. When using Umo Office Viewer, you must comply with the [Umo Team Commercial Software License Agreement](./license).

## Use Cases

- Attachment preview: preview document attachments in OA/CRM/project management systems, with download/print/watermark support
- Contract viewing: controlled viewing for legal/sales contracts, with password/watermark and download/export restrictions
- Online courseware display: preview PPT/Word courseware with multi-page mode, sidebar outline, and fullscreen
- Assignment review: teachers preview student submissions and reports online, with convenient printing/export and mobile support
- Product manual preview: e-commerce manuals and installation guides, with optional download/print
- Policy publication: government sites publish regulations, notices, and guides, with share links and watermarks
- Medical reports: secure viewing of medical records and exam reports, with password and sensitive watermarking
- Financial reports: banks/brokers present reports with restricted download/export and dark theme support
- CMS document preview: preview technical docs/whitepapers/manuals before and after publishing
- Cloud storage preview: preview team files in cloud drives/collaboration platforms, with share links and metadata
- System integration: a unified preview component for enterprise systems with multi-format, i18n, and theming
