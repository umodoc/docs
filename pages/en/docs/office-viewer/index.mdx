# Introduction

Umo Office Viewer is an Office document viewer for the web. It supports previewing Microsoft Office documents, WPS Office documents, OpenDocument (ODF), Unified Office Format (UOF), and common formats such as PDF.

This plugin is commercial. When using Umo Office Viewer, you must comply with the [Umo Team Commercial Software License Agreement](./license).

## Use Cases

- Attachment preview in OA/CRM/project systems
- Contract viewing with watermark/password and download control
- Courseware preview and presentation
- File preview in internal networks (no external dependencies)

## Supported File Formats

Umo Office Viewer supports 40+ formats, including:

- Microsoft Office: Word/Excel/PowerPoint
- WPS Office: WPS/ET/DPS
- ODF: ODT/ODS/ODP/ODG
- UOF: UOT/UOS/UOP
- PDF/RTF/CSV and more
